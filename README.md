# VatStaffing Top-Down Data
Welcome to the GitHub repository for the VatStaffing Discord Bot. This repository contains all the Top-Down Data files required to correctly match sectors to their respective airports. Please note that this project is intended for development and open collaboration purposes.

Contributions are welcome and encouraged. Feel free to submit pull requests, report issues, or suggest improvements to help enhance accuracy and functionality. When contributing, please ensure that data follows the established structure and naming conventions to maintain consistency across the project.

This repository is provided “as is” and may contain incomplete or experimental data. Always validate files before deploying them in a production environment.

### Formatting:
> + Positions are truncated to the format ``ABCD_EFG`` - all central items of data are nullified, **except for** _I_TWR or _R_TWR, which maintain their full titles.
